# Module 1: Introduction to Docker

Welcome to Module 1 of the Docker Learning Path! In this module, you will learn about Docker and its core concepts.

## Learning Objectives

By the end of this module, you will be able to:

- Understand what Docker is and its purpose.
- Explain the core concepts of Docker, such as containers, images, and networks.
- Install and configure Docker on your local machine.

## Module Content

1. Introduction to Docker
   - What is Docker?
   - Why use Docker?
   - Docker components and concepts

2. Installing Docker
   - System requirements
   - Installing Docker on Linux
   - Installing Docker on macOS
   - Installing Docker on Windows

3. Docker Containers
   - What are containers?
   - Running containers
   - Managing container images
   - Container lifecycle

4. Docker Images
   - What are Docker images?
   - Creating Docker images
   - Managing Docker image layers
   - Pushing and pulling Docker images
